# Contributing to Xiaomi Data Analysis Project

Thank you for your interest in contributing! We appreciate your effort to improve this project.

## How to Contribute

### Reporting Issues
- Describe the issue clearly
- Include steps to reproduce
- Provide expected vs. actual behavior
- Share your environment details

### Submitting Changes
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Commit with clear messages
5. Push to your branch
6. Submit a pull request

## Code Guidelines

- Follow PEP 8 for Python code
- Use meaningful variable names
- Add docstrings to functions
- Keep notebooks organized and well-commented
- Test your changes before submitting

## Commit Message Format

```
[Type] Brief description

Detailed explanation (if needed)
Fixes #issue-number
```

Types: feat, fix, docs, style, refactor, test, chore

## Questions?

Feel free to open an issue or contact the project maintainers.
